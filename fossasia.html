Short description about Project:

LuCI is the main web configuration interface for OpenWrt. It is an easy way to edit the device’s configuration files. However, initially the web interface looked horribly outdated, but now we have a fresh and professional-looking theme based on twitter Bootstrap (v2.0). Now, it's high time to move the luci bootstrap theme to v3.0 and add new components with new features and update the existing ones to make it compatible.

So what's new in Bootstrap (v3.0) ?
a) Bootstrap now has a new flat design on all elements and components.
b) Bootstrap now redesigned and rebuilt to start from your handheld devices and scale up. Responsive Css is no longer separate and all responsive features are now included into the main file. That can be great for most people, but not everyone needs or wants an adaptive web site or application.
c) New powerful grid system. The new grid is significantly simplified and combined into one single grid system: the separate fluid grid system, container, and layout are removed, as they’re down to one grid.
This kitchen-sink-included pile of code will make the design and development of web interface a lot easier. It will be one responsive, device-size independent, fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases. And the four tiers of grid classes – phones, tablets, desktops, and large desktops – allow us to create complex and adaptive layouts with ease.
